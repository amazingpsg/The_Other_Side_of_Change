# 🎬 변화의 이면 (The Other Side of Change) - Hollywood Slide Deck Website

마야 샹카르(Maya Shankar)의 저서 『변화의 이면(The Other Side of Change)』 속 영웅들이 맞닥뜨린 비극과 극복의 드라마를 웅장한 할리우드 영화 포스터 스타일로 소개하는 고품격 슬라이드 덱 웹사이트입니다.

이 프로젝트는 침선비장 언년이의 **[UI 디자인 가이드라인 비법서]**를 철저히 지키며 한 땀 한 땀 바느질하여 설계되었습니다.

---

## ✨ 핵심 적용 디자인 원칙 (침선비장 언년이의 바느질)

1. **좌측 상단 조명 바이어스 원칙 (Light Source Bias from Top-Left)**
   - 카드 및 이미지 박스 좌측과 상단에 미세한 보더 하이라이트를 주고 우측 하단으로 자연스러운 3D 그림자가 깔리도록 설계하여 자연스러운 입체감을 극대화했습니다.

2. **철저한 HSL 5색 배색 가이드라인**
   - 영화관의 어둠을 표현하는 딥 차콜 배경(`hsl(240, 16%, 6%)`)
   - 웅장한 블러드 레드(`hsl(355, 100%, 50%)`)
   - 찬란한 오렌지 골드(`hsl(38, 100%, 55%)`)
   - 눈이 편안한 스노우 화이트 본문(`hsl(0, 0%, 96%)`)
   - 차분한 보조 그레이(`hsl(240, 8%, 65%)`)

3. **최적의 가독성을 위한 구조적 폭 제한**
   - 독자의 읽기 피로도를 최소화하고 집중도를 높이기 위해, 본문 컨테이너 폭을 한 줄당 35~55자로 엄격하게 제한(`max-width: 55ch`)하였습니다.

4. **피츠의 법칙 (Fitts's Law) 버튼 인터랙션**
   - 이전/다음 장표 네비게이션 버튼을 넉넉하고 시원한 크기로 설계하여 마우스 조작 피로도를 낮추고 호버 시 부드러운 스케일 업 애니메이션을 적용했습니다.

## 🚀 로컬 실행 방법

이 프로젝트는 동적 포스터 이미지 렌더링을 위해 로컬 웹 서버 환경이 필요합니다.

```bash
# 1. 프로젝트 폴더로 이동합니다.
cd change_web

# 2. 로컬 웹 서버를 구동합니다. (Python 예시)
python -m http.server 8080

# 3. 브라우저에서 아래 주소로 접속합니다.
http://localhost:8080
```
