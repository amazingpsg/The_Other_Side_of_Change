# 🎬 The Other Side of Change (영화보다 더 영화 같은 변화의 이면)

마야 샹카르의 명저 **'변화의 이면'**을 마치 할리우드 블록버스터 영화의 포스터와 같이 웅장하고 극적으로 재해석한 웹 슬라이드 덱 프로젝트입니다. 각 인물들의 인생을 뒤바꾼 극적인 변화의 순간들을 시각적인 아름다움과 함께 감상할 수 있습니다.

## ✨ 주요 기능 (Features)
- **블록버스터 시네마틱 UI**: Glassmorphism 디자인과 깊이 있는 조명 바이어스(Shadow/Highlight)를 적용하여 영화관에 온 듯한 몰입감을 제공합니다.
- **반응형 웹 디자인 (Responsive Design)**: 
  - **Desktop**: 가로 2분할의 웅장한 그리드/플렉스 레이아웃과 지능형 마우스 휠 스크롤을 지원합니다.
  - **Mobile**: 작은 화면에 맞춘 세로 1열 콤팩트 레이아웃 및 터치 스와이프(Touch Swipe) 슬라이딩을 지원합니다.
- **비주얼 다이내믹스**: CSS Transition을 활용한 부드러운 슬라이드 애니메이션과 포스터 호버(Hover) 효과가 탑재되어 있습니다.

## 🛠 사용 기술 (Tech Stack)
- **HTML5**: 시맨틱 웹 구조화
- **CSS3 (Vanilla)**: 미디어 쿼리를 통한 철저한 모바일/데스크탑 분리 및 Flexbox 레이아웃
- **JavaScript (Vanilla)**: 휠 스크롤 및 터치 이벤트 기반의 슬라이더 로직 구현
